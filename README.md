# Movie App 🎬

A movie search application built with JavaScript that allows users to search and explore movies.

## Features

- Search for movies
- Display movie information
- Show posters and details
- View movie ratings
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript
- Movie API

## What I learned

- Working with APIs
- Fetching and displaying external data
- Using asynchronous JavaScript
- DOM manipulation
- Handling user input
